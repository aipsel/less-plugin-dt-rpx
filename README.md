# less-plugin-dt-rpx
Add rpx function to support wx-mini rpx unit.

```less
.avatar {
  width: rpx(100);
  height: rpx(100);
  border-radius: rpx(10);
}
```
becomes:
```
.avatar {
  width: 100rpx;
  height: 100rpx;
  border-radius: 10rpx;
}
```

## Installation
```bash
npm i less-plugin-dt-rpx --save-dev
```
